À propos de l'application

Cette application de trading vous permet de gérer vos investissements de manière centralisée. Elle intègre plusieurs brokers (Saxo Bank, Binance) et sources de données (Yahoo Finance) pour vous offrir une vue complète de votre portefeuille.

Brokers supportés : Saxo Bank (actions), Binance (cryptomonnaies)
Sources de données : Yahoo Finance pour les informations d'entreprise
Fonctionnalités : Synchronisation automatique, passation d'ordres, suivi des performances
